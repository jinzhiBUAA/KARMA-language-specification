# 名称

OSLC Systems Modeling Language Version 2.0

# 范围

此规范建立了术语和规则，用于根据允许和必需的属性名称和值类型来定义资源。OSLC 域规范应使用这些规则和术语来描述其资源。
该规范还规定了以 RDF/XML、JSON、Atom 和 Turtle 格式创建资源表示的规则。OSLC 域规范在指定如何表示其资源时，应引用这些规则。
