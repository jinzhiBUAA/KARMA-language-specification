# About

This is a usecase for the models and ontology files used in the journal**"本体驱动的复杂产品设计模型集成"**

# Description

The metamodels and models of intelligent electric vehicle can be found in model library folder named **Intelligent Electric Vehicle**, which includes tailored modeling library for SysML and EAST-ADL.

The ontology of intelligent electric vehicle can be found under the folder **ADS**, named **Electric_Vehicle_BFO.owl**, which is based on Basic Formal Ontology (BFO) and Industrial Ontologies Foundry (IOF) Core Specification.

# Status

The models used are supported by MetaGraph version 0.5
